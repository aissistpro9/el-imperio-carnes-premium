## 🗂️ Estructura

```
el-imperio-demo/
├── index.html          # Shell principal (SPA router)
├── pages/              # Vistas inyectadas dinámicamente
├── js/                 # Lógica de la aplicación
├── css/                # Estilos personalizados
├── img/                # Recursos gráficos
├── vercel.json         # Configuración Vercel
└── netlify.toml        # Configuración Netlify
```

## ⚙️ Deploy en Vercel

1. Conecta este repositorio en [vercel.com](https://vercel.com)
2. **Root Directory**: déjalo vacío (raíz del repo)
3. Vercel detecta `vercel.json` automáticamente ✅
